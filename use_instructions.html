10. Using this software#

Tradesman is a Python package that lets you build transportation models. You can build transportation networks importing data from OSM, import political sub-divisions, population information, amenities, and building information, as well as create zones. At the end, the user will have all their data inside and AequilibraE model, which is a Python-native transport modelling software. In turn, all files used in AequilibraE are open-format (SQLite, Spatialite) and can be used by virtually any current data and GIS software/platform available.

10.1. Usage#

To get started with Tradesman, read its user reference and work through its examples repo for introductory usage demonstrations and sample code.
